# 数据格式说明

提交格式：json

文件命名：th.json

```
{
    "title": "",
    "detail": "",
    "grade": ,
    "difficulty": ,
    "idea": "",
    "answer": ""
}
```

title：标题

detail：题目内容

grade：题型等级：0（乙级），1（甲级），2（顶级）

difficulty：难度等级：0（简单），1（中等），2（困难）

idea：题目思路

answer：题目答案
